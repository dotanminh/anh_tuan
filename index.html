🧩 CORE MODULES

Chi tiết từng Module

3 module nghiệp vụ hoạt động độc lập và bổ trợ lẫn nhau

📊

Module 1: Tổng hợp báo cáo

  • Nhận file Excel/Word/PDF từ phòng ban
  • Excel: pandas.read_excel() → DataFrame
  • Word: python-docx → Text + Tables
  • PDF: PyMuPDF / pdfplumber → Text + Tables
  • Chuẩn hóa & lưu vào Supabase
  • AI phân tích, so sánh chéo giữa phòng ban
  • Xuất báo cáo tổng → Telegram / Email / Dashboard
📅

Module 2: Lên lịch & Nhắc nhở

  • User yêu cầu AI lên lịch
  • Lưu vào bảng schedules trên Supabase
  • Edge Function chạy Cron mỗi sáng 7:00
  • Query lịch hôm nay chưa nhắc
  • Gửi nhắc nhở qua Telegram Bot
  • Update reminder_sent = true
🚨

Module 3: Cảnh báo KPI

  • Data KPI parse từ Excel → Supabase
  • So sánh kpi_data vs kpi_thresholds
  • Nếu giá trị dưới ngưỡng cảnh báo → Trigger alert
  • AI phân tích nguyên nhân + Perplexity research
  • Đề xuất action items cụ thể
  • Gửi cảnh báo chi tiết qua Telegram Bot
💾 DATABASE

Database Schema

Supabase PostgreSQL — 5 bảng chính

schema.sql
-- Bảng lưu báo cáo đã parse CREATE TABLE parsed_reports ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), phong_ban TEXT NOT NULL, -- 'kinh_doanh', 'marketing', 'van_hanh' thang INTEGER NOT NULL, nam INTEGER NOT NULL, loai_bao_cao TEXT NOT NULL, -- 'excel', 'word' data_json JSONB, -- Dữ liệu đã chuẩn hóa raw_text TEXT, -- Nội dung text thô (từ Word) file_path TEXT, created_at TIMESTAMPTZ DEFAULT NOW() ); -- Bảng lịch làm việc CREATE TABLE schedules ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), title TEXT NOT NULL, description TEXT, scheduled_date DATE NOT NULL, scheduled_time TIME, priority TEXT DEFAULT 'medium', -- 'high', 'medium', 'low' assignee_lark_id TEXT, status TEXT DEFAULT 'pending', reminder_sent BOOLEAN DEFAULT FALSE, created_at TIMESTAMPTZ DEFAULT NOW() ); -- Bảng dữ liệu KPI CREATE TABLE kpi_data ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), phong_ban TEXT NOT NULL, chi_so TEXT NOT NULL, -- 'doanh_thu', 'ty_le_chuyen_doi'... gia_tri NUMERIC NOT NULL, don_vi TEXT, -- '%', 'VND', 'đơn' thang INTEGER NOT NULL, nam INTEGER NOT NULL, created_at TIMESTAMPTZ DEFAULT NOW() ); -- Bảng ngưỡng KPI CREATE TABLE kpi_thresholds ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), chi_so TEXT NOT NULL UNIQUE, nguong_tot NUMERIC NOT NULL, nguong_canh_bao NUMERIC NOT NULL, don_vi TEXT, updated_at TIMESTAMPTZ DEFAULT NOW() ); -- Bảng log cảnh báo CREATE TABLE alerts_log ( id UUID PRIMARY KEY DEFAULT gen_random_uuid(), loai TEXT NOT NULL, -- 'kpi_warning', 'schedule_reminder' noi_dung TEXT NOT NULL, phong_ban TEXT, da_gui BOOLEAN DEFAULT FALSE, created_at TIMESTAMPTZ DEFAULT NOW() );
